Quick reference
  • enable secret cisco sets the admin password.
  • line con 0 opens console line settings.
  • password cisco sets the line password.
  • login enforces the password prompt.
  • line vty 0 4 opens Telnet line settings.
  • service password-encryption encrypts line passwords.
  • end returns to privileged mode.
  • wr saves the running config.
