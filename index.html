Cisco Password Command Trainer

This training page helps you practice the basic Cisco router commands for adding an enable secret, console password, and VTY password. It is built like a lightweight lab simulator so you can drill the command flow until it becomes second nature.

Lesson and Reference

Standard password setup

conf t
enable secret cisco
line con 0
password cisco
login
line vty 0 4
password cisco
login
service password-encryption
end
wr

What each part does

enable secret = admin mode password line con 0 = console line password = sets login password login = forces password prompt line vty 0 4 = telnet lines wr = save config

Shortcuts

conf t
line con 0
pass cisco
line vty 0 4
sh run | i pass
sh ip int br
